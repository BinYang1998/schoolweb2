
1.In the top right, click Settings Settings and then Settings.
2.On the left side of the screen, click the name of the calendar you want to embed.
3.In the "Integrate calendar" section, copy the iframe code displayed. 
4.Under the embed code, click Customize.
5.Choose your options, then copy the HTML code displayed.
6.Your embedded calendar will only be visible to people you've shared it with. To allow all visitors to see your calendar, you'll need to make it public.

To let anyone save your calendar event, you'll need to make your calendar public.

1.On a public calendar, create or edit an event.
2.Click More actions and then Publish event
3.In the "Publish event" window, copy the HTML code displayed.
4.Open your website editor, then paste this code where you want the event button to display.
